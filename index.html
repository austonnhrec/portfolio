Wires are being held in the T-568B layout

Hello Copper Cable!!

I have conducted a hands-on project to create a Straight Through Cat5 copper cable in the T568B layout. The project involved using wire cutters, wire strippers, and a crimper. I achieved experience in creating a copper cable using the appropriate tools and have been able to recreate the project with effeciency and improvement each time.

Linux basics in the form of a CTF type game

Brigante is a self-paced, hands-on exercise with multiple levels designed to familiarize beginners with the Linux user interface and terminal commands. It is based on OverTheWire.org's Bandit wargame. The game is presented as a series of increasingly difficult levels. Solving one level gives the user access to the next, and students will have to apply problem-solving skills as they make their way through the series of challenges. It was quite difficult for me to get through but I persevered.

You broke it. You put it back together.

Conducted hands-on dismantling and reassembling of a desktop computer; using an anti-static wrist strap, a screwdriver, and an electromagnetic mat; while identifying each part and how each component could be exploited by an attacker. I had collaborated with one of my peers to understand each component of a computer and identify each potential security vulnerability that it may hold.
